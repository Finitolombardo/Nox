# Playbook: Notion Deduplication by Title

## Definition:
- **Duplicate:** Two tasks with the same title (trimmed, case-insensitive).

## Rules:
1. **Never delete tasks.** Only archive duplicates.
2. **Always keep the user-owned task.**
3. If ownership is unknown, keep the one that is older **OR** has more content; archive the rest.
4. **Responsible Assignment:** For the kept user task, do not change 'Responsible' unless explicitly instructed.

- **BEFORE List:**

- **Actions:** Archive the identified duplicates and note the reason for archiving each.
